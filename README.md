# Capuchin_call_classification

This project attempts to re impliment Nicholas Renotte's version of the Z by HP Unlocked Challenge 3 - Signal Processing using PyTorch instead of Tensorflow, as a learning exercise.

## The Task
The goal of the challenge is to count the number of Capuchin bird calls from a given clip.

## Status
In Progress.

##Progress
- Converting Audio Data to Tensor
- Transforming the waveform to a spectrogram
- Training the CNN model
